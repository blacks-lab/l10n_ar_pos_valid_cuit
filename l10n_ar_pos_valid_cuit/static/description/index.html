Argentina POS CUIT Validator

Módulo para validar números de CUIT/CUIL argentinos en el Point of Sale de Odoo 17.

Validación Automática
Valida automáticamente los CUITs al crear o editar partners en el POS.
Algoritmo Oficial
Utiliza el algoritmo oficial de validación de dígito verificador argentino.
Formatos Flexibles
Acepta CUITs con guiones, puntos, espacios o sin separadores.
Mensajes Detallados
Proporciona mensajes de error específicos y detallados.
